# 状态机

状态：
- LOBBY
- GAME_INIT
- ROUND_SPEAKING
- ROUND_VOTING
- ROUND_TIE_BREAK
- ROUND_GUESSING
- ROUND_RESULT
- GAME_FINISHED

当前实现迁移：
- ROUND_SPEAKING -> ROUND_VOTING
- ROUND_VOTING -> ROUND_TIE_BREAK（平票）/ ROUND_GUESSING（非平票）
- ROUND_TIE_BREAK -> ROUND_GUESSING
- ROUND_GUESSING -> ROUND_RESULT
- ROUND_RESULT -> ROUND_SPEAKING（下一轮）
- 任意阶段 -> GAME_FINISHED（主持人结束或猜词命中）

约束：
- 仅主持人可推进阶段
- 非法阶段提交投票/猜词直接拒绝
- 出局玩家不可继续投票/猜词
